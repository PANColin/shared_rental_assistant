合租助手 - MVP产品方案
一、项目核心想法
做一个“合租小助手”工具型平台。
前期通过租房工具和租房内容吸引用户，
后期连接房东、中介和房源实现商业化。
核心模式：工具获取用户 → 内容建立信任 → 房源变现。
二、产品定位
目标用户：
1. 刚毕业的年轻人
2. 在城市打工的租房人群
3. 找合租的人
4. 第一次租房的小白

核心痛点：
1. 不会找靠谱房子
2. 容易被黑中介或假房源坑
3. 不知道如何分摊合租费用
4. 不知道租房合同注意事项
5. 不知道如何和室友管理公共费用
三、MVP最小产品功能
MVP阶段只做4个功能模块：

1. 合租费用计算
2. 租房避雷
3. 租房指南
4. 实用工具

页面结构：
首页
 ├ 合租费用计算
 ├ 租房避雷
 ├ 租房指南
 └ 实用工具
四、核心功能：合租费用计算器
输入信息：
1. 总房租
2. 室友人数
3. 每个人居住天数
4. 水电燃气等公共费用

点击计算后：
系统自动计算每个人需要支付的费用。

用途：
1. 合租房租分摊
2. 水电费分摊
3. 按居住天数分摊
4. 过年回家等特殊情况费用计算
五、租房避雷模块
提供常见租房骗局和避坑信息：

示例内容：
1. 黑中介套路
2. 二房东骗局
3. 假房源识别
4. 押金不退怎么办
5. 租房合同注意事项
六、租房指南
为租房新人提供教程：

内容示例：
1. 第一次租房完整流程
2. 如何找到靠谱合租室友
3. 租房合同怎么签
4. 押金如何防坑
七、工具合集
后期可以增加一些实用工具：

1. 合租费用计算器
2. 租房预算计算器
3. 室友值班表
4. 搬家清单
5. 合租协议模板
八、技术实现方案
方案一（推荐）：
纯前端H5项目

优点：
1. 不需要服务器
2. 部署简单
3. 成本低

部署平台：
Vercel / Cloudflare Pages

方案二：
使用uni-app开发，同时发布H5和小程序。
九、流量获取方式
1. 租房微信群
分享合租费用计算器

2. 小红书
发布租房攻略 + 工具推荐

3. 知乎
写租房经验文章

4. 抖音
发布租房经验短视频
十、后期升级方向
当有一定用户后，可以增加：

1. 房源推荐
2. 房东直租
3. 中介发布房源
4. 房源推广
十一、变现方式
1. 房源推广
房东或中介付费推广房源

2. 广告
搬家公司、保洁、宽带、家电

3. 会员服务
提供完整租房攻略、合同模板等
